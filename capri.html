Capri

Bring your favorite Windows cursor themes to Linux

Capri is a small Avalonia app that converts Windows cursor packs (.cur / .ani) into the Xcursor format and installs them as your active cursor theme on Pop!_OS / COSMIC — no terminal required.

What Capri does
Six things that make cursor installs painless
One-click install
Point Capri at a folder of .cur / .ani files and apply them as your system cursor theme instantly.
Auto mode
Capri automatically matches files to the right cursor role — pointer, wait, text, resize and more — using smart name heuristics.
Manual mode
Don't trust the guesswork? Assign each cursor role to a specific file yourself from a simple dropdown list.
Reset to system
Restore your previous cursors at any time — Capri backs up every file it replaces before touching it.
Full alias coverage
Capri automatically creates the symlinks apps expect, so cursors look correct across GTK, Qt and Electron apps.
Lightweight & native
Built with Avalonia, Capri fits right into the COSMIC desktop and feels at home there.
Why people install Capri
No terminal required
Browse a folder, pick a mode, click apply — the whole workflow lives in the app window.
Nothing lost
Every file Capri replaces is backed up first, and Reset to System can undo the whole change.
Works beyond COSMIC
Any Xcursor-based desktop works — GNOME, KDE and others, not just Pop!_OS.
